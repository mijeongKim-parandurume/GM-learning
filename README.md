## 📝 실습 가이드

### 실습 1: NotebookLM으로 공문 분석하기

**사용 도구:** [Google NotebookLM](https://notebooklm.google.com)

**실습 시나리오:**
2024년 동절기 지원 사업 공문이 도착했습니다. 작년 지침과 무엇이 달라졌는지, 당장 무엇을 준비해야 하는지 빠르게 파악해야 합니다.

**실습 파일:**
- `공문_2024_동절기지원.pdf` - 올해 공문
- `공문_2023_동절기지원(참고용).pdf` - 작년 공문
- `내부_회의록_11월.txt` - 내부 회의 내용

**실습 단계:**

1. **NotebookLM 접속**
   - https://notebooklm.google.com 접속
   - Google 계정으로 로그인
   - "새 노트북" 생성

2. **파일 업로드**
   - 3개의 실습 파일을 모두 업로드
   - 파일이 처리될 때까지 대기

3. **프롬프트 실습**

   **[프롬프트 1] 핵심 내용 3줄 요약**
   ```
   업로드된 '공문_2024_동절기지원.pdf'의 내용을 바탕으로,
   핵심 지원 대상, 필수 제출 서류, 마감 기한을 포함하여
   3줄로 요약해 줘.
   ```

   **[프롬프트 2] 변동 사항 비교 분석**
   ```
   작년 공문(2023)과 비교했을 때,
   올해 지원 금액이나 절차에서 달라진 점이 무엇인지
   표로 정리해서 보여줘.
   ```

   **[프롬프트 3] 담당자 할 일(To-Do) 추출**
   ```
   이 공문을 처리하기 위해
   실무 담당자가 이번 주 내로 해야 할 일을
   우선순위대로 리스트업 해 줘.
   ```

**학습 포인트:**
- AI는 여러 문서를 동시에 분석할 수 있습니다
- 구체적인 질문을 할수록 정확한 답변을 얻습니다
- 비교, 요약, 추출 등 다양한 작업이 가능합니다

### 실습 2: AI Studio로 보고서 자동 생성하기

**사용 도구:** [Google AI Studio](https://aistudio.google.com)

**실습 시나리오:**
10월 한 달간의 기부 및 배분 실적 데이터가 있습니다. 이를 기관 공식 양식인 '월간 운영 보고서' 포맷에 맞춰 작성해야 합니다.

**실습 파일:**
- `10월_기부실적_Raw.txt` - 정리되지 않은 원본 실적 데이터
- `월간보고서_양식.txt` - 기관 공식 보고서 양식

**실습 단계:**

1. **AI Studio 접속**
   - https://aistudio.google.com 접속
   - Google 계정으로 로그인
   - "새 프롬프트" 시작

2. **파일 내용 확인**
   - `10월_기부실적_Raw.txt` 파일 열어서 내용 확인
   - `월간보고서_양식.txt` 파일 열어서 양식 확인

3. **프롬프트 작성 및 실행**

   ```
   너는 광명 푸드뱅크의 10년 차 행정 전문가야.
   아래 제공된 [10월 실적 데이터]를 분석해서,
   [월간 보고서 양식]에 맞춰 초안을 작성해 줘.

   작성 조건:
   1. 수치는 정확하게 기입할 것.
   2. '성과 분석' 항목에는 전월 대비 긍정적인 변화를 강조할 것.
   3. 톤앤매너는 공적이고 전문적인 어조를 유지할 것.

   [10월 실적 데이터]
   (여기에 10월_기부실적_Raw.txt 내용 복사)

   [월간 보고서 양식]
   (여기에 월간보고서_양식.txt 내용 복사)
   ```

4. **결과 검토 및 수정**
   - AI가 생성한 보고서 초안 확인
   - 필요시 추가 수정 요청
   - 최종 보고서 복사하여 활용

**학습 포인트:**
- AI에게 역할을 부여하면 더 나은 결과를 얻습니다
- 구체적인 작성 조건을 명시하면 원하는 형식으로 결과를 받습니다
- 데이터와 양식을 함께 제공하면 자동으로 채워집니다

## 💡 프롬프트 작성 팁

### 효과적인 프롬프트 구성 요소

1. **역할 부여**
   ```
   너는 [직무/전문가]야.
   예: 너는 푸드뱅크의 10년 차 행정 전문가야.
   ```

2. **구체적인 작업 지시**
   ```
   [데이터]를 분석해서 [형식]에 맞춰 [결과물]을 만들어 줘.
   ```

3. **작성 조건 명시**
   ```
   작성 조건:
   1. 조건 1
   2. 조건 2
   3. 조건 3
   ```

4. **데이터 제공**
   ```
   [데이터명]
   (실제 데이터 내용)
   ```

### 자주 사용하는 프롬프트 패턴

| 목적 | 프롬프트 예시 |
|------|---------------|
| **요약** | "다음 문서를 3줄로 요약해 줘" |
| **비교** | "A와 B를 비교하여 차이점을 표로 정리해 줘" |
| **추출** | "다음 문서에서 [항목]만 리스트로 추출해 줘" |
| **생성** | "[양식]에 맞춰 [데이터]를 채워서 문서를 작성해 줘" |
| **분석** | "다음 데이터를 분석하고 인사이트를 제공해 줘" |

## 🎯 실무 적용 예시

### 1. 매일 쏟아지는 공문 처리
- **Before:** 공문 10장을 읽고 요점 정리하는데 2시간 소요
- **After:** NotebookLM에 업로드 후 5분 내 핵심 파악
- **절감 시간:** 약 1시간 55분

### 2. 월간 보고서 작성
- **Before:** 데이터 정리하고 보고서 작성하는데 2일 소요
- **After:** AI Studio로 초안 생성 후 검토만 1시간
- **절감 시간:** 약 15시간

### 3. 회의록 기반 할 일 정리
- **Before:** 회의록 읽고 수작업으로 To-Do 정리 30분
- **After:** NotebookLM에 질문하여 우선순위별로 정리 3분
- **절감 시간:** 약 27분

## ❓ FAQ

**Q1. AI 도구는 유료인가요?**
A. NotebookLM과 AI Studio는 기본적으로 무료로 사용 가능합니다. 다만 일일 사용량 제한이 있을 수 있습니다.

**Q2. 개인정보가 포함된 문서를 업로드해도 되나요?**
A. 민감한 개인정보는 가명 처리하거나 샘플 데이터로 대체하는 것을 권장합니다.

**Q3. AI가 생성한 내용을 그대로 사용해도 되나요?**
A. AI는 초안 작성 도구로 활용하고, 반드시 실무자가 최종 검토 및 수정해야 합니다.

**Q4. 오프라인에서도 사용할 수 있나요?**
A. NotebookLM과 AI Studio는 인터넷 연결이 필요합니다.

**Q5. 강의 자료를 다른 기관과 공유해도 되나요?**
A. 본 자료는 교육 목적으로 자유롭게 활용 가능합니다.
