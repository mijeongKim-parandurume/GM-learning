Practice 01

NotebookLM: 공문 지식 비서 만들기

수십 장의 공문을 다 읽지 마세요. AI에게 업로드하고 물어보면 됩니다.

Tool Comparison

똑똑한 박사님과
꼼꼼한 비서의 차이

모든 것을 아는 AI가 항상 정답은 아닙니다.
우리 푸드뱅크 업무에는 '내부 문서를 아는' AI가 필요합니다.

VS
ChatGPT, Gemini, Claude

일반 생성형 AI

"인터넷의 방대한 지식을 학습한 똑똑한 박사님"

지식의 출처 (Source)
🌐 전 세계 인터넷 데이터
치명적 단점 (Risk)
⚠️
환각 (Hallucination) 모르는 내용도 그럴듯하게 지어낼 수 있음
추천 업무

  • 일상적인 이메일 초안 작성
  • 번역 및 일반 상식 검색

NotebookLM

"우리 기관 문서를 달달 외운 꼼꼼한 비서"

지식의 출처 (Source)

  • 회의록 요약 및 할 일 추출

왜 NotebookLM이어야 할까요?

🤥
거짓말 제로

일반 AI는 2023년 자료를 섞어서 답할 수 있지만,
NotebookLM은 내가 올린 파일 안에서만 답을 찾습니다.

📌
확실한 근거(Citations)

답변 끝에 달린 [1] 숫자를 누르면
PDF 원문의 해당 위치로 즉시 이동합니다.

📚
대용량 통합 분석

수십 개의 공문과 회의록을 한 번에 올려두고
"전체 내용을 종합해줘"라고 명령할 수 있습니다.

📂 실습 시나리오 및 파일

2024년 동절기 지원 사업 지침이 내려왔습니다. 작년 지침과 무엇이 달라졌는지, 당장 무엇을 준비해야 하는지 파악해야 합니다.

Required Files

📄 공문_2024_동절기지원.pdf
📄 공문_2023_동절기지원(참고용).pdf
📝 내부_회의록_11월.txt
1

핵심 내용 3줄 요약

"업로드된 '공문_2024_동절기지원.pdf'의 내용을 바탕으로, 핵심 지원 대상, 필수 제출 서류, 마감 기한을 포함하여 3줄로 요약해 줘."
2

변동 사항 비교 분석

"작년 공문(2023)과 비교했을 때, 올해 지원 금액이나 절차에서 달라진 점이 무엇인지 표로 정리해서 보여줘."
3

담당자 할 일(To-Do) 추출

"이 공문을 처리하기 위해 실무 담당자가 이번 주 내로 해야 할 일을 우선순위대로 리스트업 해 줘."
Step-by-Step Manual

NotebookLM 사용 가이드

수십 페이지의 공문, 복잡한 매뉴얼을 다 읽지 마세요.
업로드하고 물어보면, 출처와 함께 답변합니다.

NotebookLM 접속하기 (notebooklm.google.com)

1 화면 구조 익히기 (책상 정리법)

새 노트북(New Notebook)을 만들면 화면이 크게 왼쪽과 오른쪽으로 나뉩니다.

왼쪽 (Source)
자료실 (업로드)

공문, PDF, 텍스트 파일 등을
여기에 추가(+)합니다.
(AI가 공부할 교과서)

오른쪽 (Chat)
대화방 (질문)

업로드된 자료를 바탕으로
질문하고 답변을 받습니다.
(AI 비서와의 채팅창)

2 따라하기: 공문 분석 실습

1
새 노트북 만들기

메인 화면에서 [New Notebook] 박스를 클릭하고,
제목을 "동절기 지원 사업 분석"이라고 입력하세요.

2
자료(Source) 업로드

왼쪽 패널의 [+] (Add source) 버튼을 누르세요.
제공해드린 실습 파일 3개를 드래그해서 업로드합니다.

📄 공문_2024_동절기_지원사업_안내.txt
📄 공문_2023_동절기_지원사업_안내_참고용.txt
📝 내부_11월_주간회의록.txt
3
질문하기 (통합 분석)

오른쪽 채팅창에 아래 질문을 입력해보세요.

"2023년 공문(참고용)과 비교했을 때, 올해(2024년) 지원 사업에서 가장 크게 달라진 점 3가지를 표로 정리해 줘."

"내부 회의록 내용을 참고했을 때, 담당자가 가장 걱정하는 문제점이 무엇이고, 이를 해결하기 위해 공문에서 확인해야 할 사항은 뭐야?"

4
팩트 체크 (Citation 확인)

AI 답변 끝에 있는 작은 1 2 숫자를 클릭해보세요.
왼쪽 화면에서 AI가 참고한 문장의 위치를 하이라이트해서 보여줍니다.

※ 숫자를 눌러 원문을 확인하는 습관이 '행정 실수'를 막아줍니다!

Practice 02

AI Studio: 보고서 자동 생성

데이터만 넣으세요. 문장은 AI가 다듬고 완성합니다.

공문서 작성, 어떤 도구가 유리할까요?

일반 챗봇 AI와 Google AI Studio의 차이를 확인해보세요

VS
💬

일반 챗봇 AI

ChatGPT, Gemini (웹사이트)

  • ⚠️ 일관성 부족: 매번 말투나 형식이 조금씩 달라짐
  • ⚠️ 휘발성: 대화가 길어지면 앞의 지시를 잊어버림
  • 👍 장점: 사용이 쉽고 아이디어/초안 작성에 빠름
Best for 이메일, 아이디어 구상, 요약
⚙️

Google AI Studio

프롬프트 엔지니어링 도구

  • ✔ 완벽한 형식 유지: 예시(Few-shot)를 학습해 형식을 고정
  • ✔ 페르소나 고정: "행정 전문가" 역할을 끝까지 유지
  • ✔ 재사용성: 템플릿으로 저장해 매달 데이터만 교체
Best for 정기 보고서, 공문서, 대량 업무

💡 이번 실습에서는 AI Studio를 사용하여 '변하지 않는 나만의 보고서 봇'을 만듭니다.

📋 실습 시나리오

10월 한 달간의 기부 및 배분 실적이 담긴 텍스트 데이터가 있습니다. 이를 기관 공식 양식인 '월간 운영 보고서' 포맷에 맞춰 작성해야 합니다. 성과 분석 코멘트까지 AI가 작성하도록 합니다.

Input Data
10월_기부실적_Raw.txt
(기부: 120건, 쌀 500kg...)
Template
월간보고서_양식.txt
(1.개요 2.성과 3.계획)

Prompt Engineering

아래 내용을 AI Studio 프롬프트 창에 입력합니다.

AI Studio Prompt

"너는 광명 푸드뱅크의 10년 차 행정 전문가야.
아래 제공된 [10월 실적 데이터]를 분석해서,
[월간 보고서 양식]에 맞춰 초안을 작성해 줘.

작성 조건:
1. 수치는 정확하게 기입할 것.
2. '성과 분석' 항목에는 전월 대비 긍정적인 변화를 강조할 것.
3. 톤앤매너는 공적이고 전문적인 어조를 유지할 것."

Step-by-Step Manual

Google AI Studio 사용 가이드

복잡한 코딩 없이, 클릭 몇 번으로
우리 기관 업무에 딱 맞는 AI 도구를 설계하는 법을 배웁니다.

Google AI Studio 접속하기 (aistudio.google.com)

1 화면 구조 익히기 (3등분 법칙)

접속 후 'Create New'를 누르면 아래와 같은 화면이 나옵니다.

왼쪽 (Left)

파일 서랍

내가 만든 프롬프트 저장 목록
(My Library)

가운데 (Center)

작업 책상 (Main)

1. System Instructions (AI 성격 부여)
2. User Input (데이터 입력)

오른쪽 (Right)

조절기

모델 선택 (Gemini 1.5)
창의성 조절 (Temperature)

2 따라하기: 보고서 생성기 만들기

1

모델 선택하기 (오른쪽 패널)

오른쪽 Model 드롭다운 메뉴에서 Gemini 1.5 Flash를 선택하세요. (가장 빠르고 효율적입니다.)

2

AI의 '성격' 부여하기 (System Instructions)

화면 상단의 System Instructions 박스를 클릭하고 아래 내용을 입력하세요.
* 이 부분이 일반 챗GPT와 다릅니다. AI의 역할을 고정시키는 단계입니다.

너는 광명 푸드뱅크에서 10년 근무한 꼼꼼한 행정 전문가야.
입력된 데이터를 바탕으로 객관적이고 정중한 어조의 보고서를 작성해야 해.
없는 사실을 지어내지 말고, 수치는 정확하게 계산해서 기입해 줘.
3

데이터와 명령 입력하기 (User Input)

화면 하단의 큰 채팅창(Type something)에 실습용 데이터 파일 내용을 붙여넣으세요.

[입력 순서]
  1. 1. 월간_운영_보고서_양식.txt 내용을 붙여넣기
  2. 2. 그 아래에 10월_기부_배분_실적_RawData.txt 내용을 붙여넣기
  3. 3. 마지막 줄에 명령 입력: "위 데이터를 사용해 보고서 양식을 채워 줘."
4

실행(Run) 및 저장(Save)

Run 버튼(또는 Ctrl+Enter)을 눌러 결과를 확인하세요.
결과가 마음에 들면 상단 제목(Untitled Prompt)을 클릭해 '월간 보고서 생성기'로 저장하세요.

Pro Tip

가장 중요한 설정: Temperature (온도)

🌡️ 온도를 낮추면 (0.0 ~ 0.2)

AI가 차분하고 사실적으로 변합니다.
✅ 보고서, 공문서, 팩트 체크용

🌡️ 온도를 높이면 (0.8 ~ 1.0)

AI가 창의적이고 자유분방해집니다.
❌ 공문서 작성 시에는 비추천 (말을 지어냄)

※ 오른쪽 패널의 Temperature 바를 조절하여 0.1로 설정하세요!
